# Docker Wordpress

Projeto com configurações básicas para iniciar um projeto wordpress com docker

## Sobre o projeto

### Serviços do docker

- wordpress
  - imagem &rarr; wordpress:php8.3-apache

- db
  - imagem &rarr; mysql:8.0.36

### Como executar o projeto

```bash
docker-compose build
```

```bash
docker-compose up
```
